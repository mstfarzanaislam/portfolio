Projects

Here, some of my projects related to electronics and web development are shown.

Password based door lock system using Proteous software

A Password-Based Door Locking System in Proteus involves simulating a microcontroller-based system that controls access to a door via a keypad. The user enters a password, which is compared to a stored value in the microcontroller. If the password is correct, a servo motor or solenoid unlocks the door; if incorrect, a buzzer sounds. The system can also display feedback on an LCD screen to indicate success or failure.

Wireless audio transmission using laser beam

The Wireless Audio Transmission Using Laser Beam project involves transmitting audio signals through modulated laser light. The audio signal is first converted into an electrical signal, which modulates the laser beam. On the receiver side, a solar panel detects the modulated laser light, converting it back into an electrical signal, which is then amplified and sent to a speaker for playback. This project demonstrates wireless communication using optical signals for audio transmission.
